# sudukoSolver

Have to upload txt file of sudoku puzzle with numbers seperated by space " " and "0" for empty spaces.
Example:

0 4 0 0 0 8 0 0 0 
8 5 0 2 7 0 1 0 0 
3 0 0 0 0 0 6 7 0 
0 0 0 0 2 0 0 0 0 
0 6 1 0 0 0 0 0 5 
0 0 4 0 1 9 0 8 3 
0 0 0 4 0 0 0 1 0 
0 8 0 0 5 0 0 0 6
